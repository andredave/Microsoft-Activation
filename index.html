What is Microsoft Activation Script (MAS)?

Microsoft Activation Script (MAS) is a lightweight, safe and powerful Windows activator that allows you to activate Windows 11 without product key, activate Windows 10 permanently, and activate Microsoft Office 365, 2021, 2024. It is currently one of the best and most popular free solutions for permanent Windows activation.

Key Features

  • Permanent Windows 11 activation and Windows 10 activation
  • Full support for all editions: Home, Pro, Enterprise, Education
  • Microsoft Office activation (Office 365, 2021, 2024)
  • Multiple methods: HWID Activation, KMS Activator, KMS38, Ohook
  • Works offline after initial setup
  • No installation required • No product key needed
  • Removes "Activate Windows" watermark

How to Activate Windows 11 / Windows 10 (2026)

  1. Download the latest MAS.zip

  2. Extract the archive
  3. Temporarily disable your antivirus
  4. Run mas.exe as Administrator
  5. Select the desired activation method
⚠️ Important: Windows Defender and other antiviruses often flag mas.exe as a false positive. Disable real-time protection before running the activator, then re-enable it after successful activation.
